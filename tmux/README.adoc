= TMUX

== Install

run install script

[source, shell]
----
./install.sh
----

== Description

=== Usage

[source, shell]
----
tmux # start tmux
tmux new -s <session name> # create a new session
tmux a -t <session name> # attach to a session
tmux a # attach to the last session
tmux ls # list all sessions
pkill tmux # kill all tmux sessions
----

==== Keyboard shortcuts

|===
|shortcut |description
|kbd:[Ctrl + w]
|prefix key.
2+^.^|*split windows*
|kbd:[prefix + F12]
|vertical split window and open htop.
|kbd:[prefix + s]
|vertical split window.
|kbd:[prefix + v]
|horizontal split window.
|kbd:[prefix + >] or kbd:[<]
| resize pane to the left or right.
|kbd:[prefix + +] or kbd:[=]
| resize pane to the up or down.
|===
