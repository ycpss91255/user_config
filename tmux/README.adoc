== Usage

|===
|shortcut |description

|kbd:[Ctrl + w]
|prefix key.

2+^.^| split windows

|kbd:[prefix + F12]
|vertical split window and open htop.

|kbd:[prefix + s]
|vertical split window.

|kbd:[prefix + v]
|horizontal split window.

|kbd:[prefix + >] or kbd:[<]
| resize pane to the left or right.

|kbd:[prefix + +] or kbd:[=]
| resize pane to the up or down.

|===


|===
|1 >s|2 |3 |4
^|5 2.2+^.^|6 .3+<.>m|7
^|8
|9 2+>|10
|===
